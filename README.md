# Welcome to Kia Talk Chat App

A blazing fast Node-Express Chat Room App based on socket.io client and server libraries which can share, broadcast messages with live ..typing status.

### Concurrent Chat
### Broadcast
### See live ..typing status


### Start the project
`cd chat-room-app`
`npm install`
`npm run start`
